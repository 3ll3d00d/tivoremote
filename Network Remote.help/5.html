Command-Line Options

-t, --force-tk
Use the Tkinter GUI even if PyGTK is available.
-l, --landscape
Move the second half of the button display to a position to the right of the first half, instead of below it. The default layout is similar to a real TiVo peanut, which makes for a very tall, narrow window -- too tall for some environments.
-h, --help
Print help and exit.
-k, --keys
Print the list of keyboard shortcuts and exit.
-v, --version
Print the version and exit.
-z, --nozeroconf
Don't try the Zeroconf-based method of detecting TiVos.
-g, --graphics
Use "graphical" labels for some buttons.
<address>
Any other command-line option is treated as the IP address (name or numeric) of the TiVo to connect to. Connection is automatic on startup, and disconnection on exit.
